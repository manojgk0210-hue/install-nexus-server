# Nexus Repository Installation

This repository contains a shell script to install and configure Sonatype Nexus Repository Manager on Ubuntu.

## Files

- `install-nexus.sh` – Shell script to install and configure Nexus.
- `README.md` – Installation instructions.

## Usage

```bash
sudo nano install-nexus.sh
sudo chmod +x install-nexus.sh
sudo ./install-nexus.sh
